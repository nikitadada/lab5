Данный проект позволяет работать с уравнениями гиперплоскости в одно-, двух-, трех- и четырехмерном пространствах. Существует возможность создавать уравнения любой гиперполоскости в произвольном порядке, позже их можно отсортировать. Также есть возможность выполнять перенос, а также выполнять симметрию.Есть метод, который для каждого уравнения может возвратить размерность пространства.
